======================== JQuery and Commander REST API examples =======================

What is it?
__________

This folder contains 3 main things:

1. commanderREST.html - This is the main file that has all the example code in it.  Run this by opening it up in your favorite browser

2. jquery.*.js files - These files make up the jquery javascript library. This library is used for javascript abstraction and to make all the AJAX calls.  Most jquery code is prefixed with a '$'. For info see: http://jquery.com/

3. flot directory - This folder contains the javascript library for creating javascript only graphs/charts.  For more info check out: http://www.flotcharts.org/

4. images directory - Supporting images for jquery UI stuff.
